To run this application on a server, like srcappstest01:

1.) Get a command prompt
2.) CD to D:\Executables\Ecan.WaterUseAlerts.LoadFile
3.) Run Ecan.WaterUseAlerts.LoadFile.exe and enter the date in dd-mmm-yyyy format
        example: Ecan.WaterUseAlerts.LoadFile.exe 29-jan-2019
4.) Erros will be reported in D:\Executables\Ecan.WaterUseAlerts.LoadFile\Log\DailyAlertLoad.log file
5.) SQL Servers are sql2012dev01 and sql2012test01
6.) The table updated is in the Hilltop database, DailyAlert
